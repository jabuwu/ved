ved is dev backwards and is only meant for development

ved is my attempt to shift all my local development scripts to deno

install ved using deno: 
deno install -A --unstable -n ved --no-check -r @ORIGIN@/cli.ts
or using the easier to remember: 
curl @ORIGIN@ | sh
create a github repository with deno scripts including folders with index.ts files inside 
mycommand/index.ts
myothercommand/index.ts
initialize with a github repository: 
ved --init myname/ved-scripts
run the commands: 
ved mycommand
ved myothercommand
